# ESP32 Blink

This is a simple example project that makes the builtin LED of the _DOIT ESP32-WROOM-32D_ development board blink.

## Requirements
- VsCode & PlatformIO
- [USB to UART driver](https://www.silabs.com/developers/usb-to-uart-bridge-vcp-drivers)
